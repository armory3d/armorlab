! This is an experimental build for Android !

Recommended:
- Android: 10.0 or newer
- GPU: Adreno 530 / Mali T880MP12 or faster
- CPU: 64bit (arm64-v8a)

Installation:
- Copy .apk file to Android device
- Open the .apk with file browser to install it
- Allow installing from unknown sources

This build is only intended for testing and
may be sub-par in other areas compared to regular builds.

Please report bugs at:
https://github.com/armory3d/armorlab/issues
